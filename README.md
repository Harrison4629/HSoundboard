### HSoundboard

A discord-like soundboard for windows

### Build

1.git clone

```bash
   git clone https://github.com/Harrison4629/HSoundboard.git
```

2.install environment

```bash
   cd HSoundboard
```

```bash
   npm install
```

3.build

run dev:

```bash
   npm run dev
```

run build:

```bash
   npm run build:win
```
